## 🚀 Quick start

### Installation

```bash
$ cd be-the-hero
$ cd frontend && yarn
```

```bash
$ cd .. && cd backend
$ yarn
```

### Usage

```bash
$ cd frontend
$ yarn start
```

> Abra outro terminal na mesma pasta e rode

```bash
$ cd backend
$ npx knex migrate:latest
$ yarn dev
```
